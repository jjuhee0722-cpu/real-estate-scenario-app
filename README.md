# 부동산 주거 시나리오 기회비용 비교 계산기

화정동 아파트 단독 매수(시나리오 A)와 남자친구 집 전세금 반환 후 입주(시나리오 B)의 자금계획, 대출한도, 월 상환 부담을 비교하는 Streamlit 앱입니다. 시나리오 A는 향후 신생아 특례대출 대환 시뮬레이션도 제공합니다.

## 설치 및 실행

Python 3.9 이상을 권장합니다.

```bash
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install streamlit pandas pytest
streamlit run app.py
```

## 휴대폰에서 접속

프로젝트에는 같은 Wi-Fi의 휴대폰에서 접속할 수 있도록 `.streamlit/config.toml` 설정이 포함되어 있습니다. Mac과 휴대폰을 같은 Wi-Fi에 연결한 뒤 앱을 실행합니다.

```bash
.venv/bin/streamlit run app.py
```

Mac의 Wi-Fi IP 주소를 확인합니다.

```bash
ipconfig getifaddr en0
```

휴대폰 브라우저에서 `http://확인한-IP:8501` 형식으로 접속합니다. 예를 들어 IP가 `192.168.45.248`이면 `http://192.168.45.248:8501`입니다.

- 앱을 실행한 Mac이 켜져 있고 잠자기 상태가 아니어야 합니다.
- Mac과 휴대폰이 같은 Wi-Fi에 연결되어 있어야 합니다.
- macOS 방화벽 확인 창이 나타나면 Python 또는 Streamlit의 수신 연결을 허용해야 합니다.
- 이 주소는 같은 로컬 네트워크용이며 인터넷 공개 주소가 아닙니다.

## 고정 공개 주소 배포

같은 Wi-Fi가 아닌 곳에서도 접속하려면 GitHub 저장소를 Streamlit Community Cloud에 연결해 배포합니다. 저장소 루트의 `requirements.txt`가 배포 환경의 Python 패키지를 설치합니다.

1. 이 프로젝트를 GitHub 저장소에 올립니다.
2. [Streamlit Community Cloud](https://share.streamlit.io/)에서 GitHub 계정으로 로그인합니다.
3. **Create app**을 선택하고 저장소, `main` 브랜치, `app.py`를 지정합니다.
4. 원하는 고정 서브도메인을 선택하고 배포합니다.

배포 후에는 `https://선택한이름.streamlit.app` 형태의 고정 주소로 PC와 휴대폰에서 접속할 수 있습니다.

macOS iCloud Drive에서 Python 캐시 권한 오류가 발생하면 다음처럼 실행할 수 있습니다.

```bash
PYTHONPYCACHEPREFIX=/tmp/python-cache streamlit run app.py
```

## 테스트 실행

프로젝트 폴더에서 다음 명령을 실행합니다.

```bash
python3 -m pytest -v
```

프로젝트의 가상환경을 사용하는 경우에는 다음 명령으로 실행할 수 있습니다.

```bash
.venv/bin/python -m pytest -v
```

테스트는 원리금균등 월 상환액, 잔여 원금, DSR 대출한도, 시나리오 부족·여유자금, 신생아 특례대출 대환 계산과 0% 금리·0원 대출·대출기간 이후 대환 등의 경계값을 검증합니다.

## 입력 단위

모든 금액은 **만원** 단위입니다. `4,400`은 4,400만 원, `60,000`은 6억 원을 뜻합니다. 결과 화면은 만원과 억 원을 함께 표시합니다.

## 주요 계산 로직

- 실제 총 필요자금 = 총 필요자금 + 부대비용 + 기타비용
- 총 현금 투입액 = 내 현금 + 남자친구 현금
- 부족자금 = 실제 총 필요자금 - 총 현금 투입액 - 필요 대출금
- LTV 한도 = 총 필요자금(담보가치 가정) × 적용 LTV
- DSR 한도 = `(연소득 × DSR - 기존 부채의 연간 부담)`을 월 상환 가능액으로 바꾼 뒤 대출원금을 역산
- 최종 한도 = LTV 한도, DSR 한도, 정책 최대한도(해당 시) 중 최솟값

원리금균등상환의 월 납입액을 계산하며 금리 0%, 대출 0원, 소득 0원도 처리합니다. 스트레스 DSR을 켜면 실제 금리에 설정된 스트레스 금리를 더해 DSR 한도를 계산합니다. 기존 부채의 연간 부담은 기본적으로 잔액의 10%로 단순 추정합니다.

## 신생아 특례대출 대환 시뮬레이션

기존 대출을 입력한 대환 시점까지 원리금균등으로 갚았다고 보고 잔여 원금을 계산합니다. 잔여 원금, 중도상환수수료, 기타비용을 합친 금액을 정책 최대한도 내에서 새 대출원금으로 삼아 월 상환액과 총상환액을 다시 계산합니다. 한도 초과분은 별도 필요자금으로 표시합니다. 이자 절감액과 한도 초과 현금까지 반영한 총상환 차이는 구분해서 보여줍니다.

소득·자산·주택가격·LTV·DTI·정책 최대한도 조건을 점검하지만, 출산 요건 자체는 사용자가 ‘대환 적용’을 선택한 것으로 가정합니다.

## 중요한 안내

코드의 정책·금리 값은 요청서의 예시를 옮긴 **의사결정 보조용 기본값**입니다. 실제 대출 조건, 규제지역, DSR 산정 방식, 우대금리, 자산 및 소득 기준은 신청 시점에 달라질 수 있으므로 주택도시기금과 취급은행의 최신 기준을 반드시 확인하세요.
